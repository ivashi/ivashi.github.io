try it out

Interactive music exploration tool in the browser. Features a keyboard with adjustable tuning, scale browser, harmonic interval visualizer, 16-step sequencer, and a bounce mode that triggers notes on a spinning circle of fifths. Supports historical temperaments (Pythagorean, Meantone, Werckmeister) and microtonal systems (19-TET, 31-TET). All synthesis via the Web Audio API — no external audio libraries.
